# Todoist

## 这是什么

这个是对于国外的todoist网站的仿站练习，暂时只考虑做基础功能，不考虑高级功能。并且由于是静态页面，所以数据库暂时只能使用indexedDB，这个只能存储在本地，但是只是为了练习js。

暂时不想要的功能：
    1.  拖动
    2.  标签
    3.  过滤器
